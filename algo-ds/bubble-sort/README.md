# Сортировка пузырьком (Bubble Sort)

**Сортировка пузырьком** — это один из самых простых и известных алгоритмов сортировки. Его суть заключается в повторяющемся проходе по массиву, сравнении соседних элементов и их обмене, если они стоят в неправильном порядке.

## 🚀 Алгоритм работы

1. **Начало**: Проход начинается с начала массива.
2. **Сравнение**: Сравниваются два соседних элемента — текущий и следующий.
3. **Обмен**: Если текущий элемент больше следующего, они меняются местами.
4. **«Всплытие»**: В результате первого прохода самый большой элемент «всплывает» в самый конец массива.
5. **Повторение**: Процесс повторяется, пока на очередном проходе не окажется, что обмены больше не требуются.

## 📊 Сложность $O(n^2)$

*   **Худший и средний случай**: $O(n^2)$ — когда массив отсортирован в обратном порядке.
*   **Лучший случай**: $O(n)$ — если массив уже отсортирован.

## 💡 Особенности
Алгоритм считается неэффективным для больших массивов, однако он прост в реализации и **стабилен**.
