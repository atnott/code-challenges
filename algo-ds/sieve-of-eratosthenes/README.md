# Решето Эратосфена (Sieve of Eratosthenes) 

**Решето Эратосфена** — это классический алгоритм для поиска всех простых чисел до заданного числа $n$. Один из самых эффективных способов решения этой задачи для небольших и средних диапазонов.

## 🚀 Алгоритм работы

1. **Инициализация**: Выписать все целые числа от $2$ до $n$.
2. **Выбор первого числа**: Начать с $p = 2$ (первое простое число).
3. **Вычеркивание**: Зачеркнуть в списке все числа, кратные $p$ (начиная с $p^2, p^2 + p, \dots$).
4. **Повтор**: Найти первое невычеркнутое число больше $p$, присвоить его $p$ и вернуться к шагу 3.
5. **Завершение**: Повторять, пока $p^2 \le n$. Все оставшиеся числа — простые.

## 📊 Сложность $O(n \log \log n)$

## 💡 Особенности
Алгоритм эффективен, так как на каждом шаге мы не проверяем число на простоту, а просто «вычеркиваем» составные числа, обращаясь к ним по индексу.
