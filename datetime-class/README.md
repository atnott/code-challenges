# DateTime Class

C++ класс для удобной работы с датой и временем. Основная особенность — использование алгоритмов **Юлианских дней (JDN)** для максимально точных арифметических вычислений.

## Основные возможности

- **Умный парсинг**: Конструктор принимает строку в формате ISO 8601 (`yyyy-MM-ddThh:mm:ss`).
- **Валидация данных**: Автоматическая проверка корректности даты (дни в месяцах, високосные годы).
- **Арифметика дат**:
    - Сложение/вычитание дней (включая дробные части, например, +0.5 дня = +12 часов).
    - Нахождение разницы между двумя датами в днях.
- **Сравнение**: Полная поддержка операторов `<`, `>`, `==`, `!=`.
- **День недели**: Определение дня недели (возвращает текстовое название на русском языке).
- **Красивый вывод**: Поддержка оператора `<<` для вывода даты в формате "1 января 2026 года".
