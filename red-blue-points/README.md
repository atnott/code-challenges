# Red-Blue Points Matching

Программа для решения задачи поиска оптимального соединения красных и синих точек на плоскости.

## 📝 Описание задачи

На плоскости даны координаты $n$ **красных** точек и $n$ **синих** точек.
Необходимо соединить попарно точки разного цвета отрезками так, чтобы **суммарная длина всех отрезков была наименьшей**.
