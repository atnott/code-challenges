# atnott::shared_ptr

Минималистичная реализация умного указателя с подсчетом ссылок на C++, аналогичная `std::shared_ptr`.

## Особенности
- **Подсчет ссылок**: Управляет временем жизни объекта, удаляя его только тогда, когда последняя копия `shared_ptr` выходит из области видимости.

## Ограничения (Важно!)
Данная реализация предназначена **только для одиночных объектов** (выделенных через `new`).
⚠️ **Не используйте для массивов** (`new[]`), так как деструктор вызывает обычный `delete`, что приведет к неопределенному поведению.
